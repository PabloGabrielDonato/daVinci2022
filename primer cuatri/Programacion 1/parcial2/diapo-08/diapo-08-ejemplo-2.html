Diapo 08: Arrays II - Ejemplo 2

Soda Stereo

Creamos un array multidimiensional con los integrantes de Soda Stereo e información sobre ellos.

Recorremos la matriz para mostrar la información de cada uno.

Vamos a necesitar utilizar un for dentro de otro.

Utilizaremos la variable i para recorrer el primer nivel (cada músico).

Utilizaremos la variable j para recorrer el segundo nivel (cada dato del músico).

Recordemos que para ver el dato de cada músico, necesito la coordenada (músico-dato), es decir [i][j].
